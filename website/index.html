Commands Guide

Designed for developers, TunnelX offers both a rich interactive UI and scriptable headless modes.

Basic Usage
# Expose a local HTTP server on port 8080
tunnelx http 8080

# Request a specific subdomain
tunnelx http 3000 --subdomain my-api

# Expose a local directory directly
tunnelx file --path ./public

# Forward raw TCP traffic (e.g., SSH, Database)
tunnelx tcp 5432
